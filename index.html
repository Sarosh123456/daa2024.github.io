Subject: Design and Analysis of Algorithm

Project Overview

In this project, I have explored various concepts of Design and Analysis of Algorithms. The goal was to apply algorithm design techniques such as divide and conquer, dynamic programming, and greedy algorithms to solve different computational problems efficiently. Below are some key points of the project:

  • Analysis of algorithm performance (time and space complexity)
  • Application of sorting algorithms like Merge Sort and Quick Sort
  • Greedy algorithms for optimization problems
  • Dynamic programming for solving recursive problems efficiently
